**Repository of dotfiles**

This repository includes custimized config files for various programs.

The intent is to be able to quickly and easily get up and running should I need to setup a new machine.

## Usage

Clone this repository and use the `stow` command to symlink the configs you want to use.

```bash
git clone https://github.com/Zerossive/dotfiles-olympus.git
cd dotfiles-olympus
stow bin bash ...
```
